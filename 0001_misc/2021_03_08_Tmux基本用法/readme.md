目前常用的命令如下：  
```
tmux：自动创建一个会话并进入会话  
ctrl+b+%：纵向分割窗口  
ctrl+b+"：横向分割窗口  
ctrl+b+o：跳到下一个窗格  
ctrl+b+c：创建新窗口  
ctrl+b+w：以菜单方式显示及选择窗口  
ctrl+b+[：进入copy mode，能够查看历史记录  
ctrl+b+d：退出但不关闭会话  
tmux attach：进入之前的会话  
```
